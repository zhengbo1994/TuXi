JSTS

JSTS是一个符合OGC规范的简单要素空间位置判定函数JavaScript库，JSTS也是Java类库JTS的一个接口，且与OpenLayer3具有互操作性，与NTS用法类似

示例

注意: 示例结合OpenLayer显示

读取A，B两个点的WKT格式

var reader = new jsts.io.WKTReader()
var a = reader.read('POINT (-20 0)')
var b = reader.read('POINT (20 0)')

A，B缓冲区分析

a = a.buffer(40)
b = b.buffer(40)

A，B相交分析

var intersection = a.intersection(b)

A异同B

var difference = a.difference(b)

A合并B

var union = a.union(b)

A,B对称异同

var symDifference = a.symDifference(b)
